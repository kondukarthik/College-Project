# Simple Calculator

## Overview

`SimpleCalculator` is a basic arithmetic calculator application built using Java Swing. It supports fundamental operations such as addition, subtraction, multiplication, and division. The application allows for continuous operations, meaning you can chain multiple calculations without needing to clear the display after each result.

## Features

- **Basic Arithmetic Operations**: Perform addition, subtraction, multiplication, and division.
- **Continuous Calculations**: Chain multiple operations seamlessly.
- **Clear Button**: Clear the current input to start fresh.

## Requirements

- **Java Development Kit (JDK)**: Version 8 or higher.
- **Java IDE or Text Editor**: Such as IntelliJ IDEA, Eclipse, VS Code, or a terminal for compiling and running Java code.

## Installation and Setup

1. **Compile the Code**:
   - Open a terminal or command prompt.
   - Navigate to the directory where `SimpleCalculator.java` is located.
   - Compile the Java code

2. **Run the Application**:
   - Execute the compiled Java application:
     java SimpleCalculator

## Usage

1. **Entering Numbers**:
   - Click on the number buttons (0-9) to input numbers.

2. **Performing Operations**:
   - Click on the operation buttons (`+`, `-`, `*`, `/`) to set the desired arithmetic operation.

3. **Calculating Results**:
   - After entering numbers and selecting an operation, click the `=` button to display the result.

4. **Clearing Input**:
   - Click the `C` button to clear the current input and reset the display.

## Example
- **To Calculate 5 + 3**:
  1. Click `5`
  2. Click `+`
  3. Click `3`
  4. Click `=`
  - The display will show `8`.

- **To Calculate 8 - 2 * 3**:
  1. Click `8`
  2. Click `-`
  3. Click `2`
  4. Click `*`
  5. Click `3`
  6. Click `=`
  - The display will show `2`.
